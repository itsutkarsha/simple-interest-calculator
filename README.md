# simple-interest-calculator

## 📌 Project Description
The Simple Interest Calculator is a basic Bash script project that calculates simple interest based on user input. It helps users quickly compute interest by entering the principal amount, rate of interest, and time period.

## ⚙️ Features
- Takes user input for:
  - Principal amount
  - Rate of interest
  - Time period
- Calculates simple interest using a standard formula
- Displays the result instantly in the terminal

## 🧮 Formula Used
Simple Interest (SI) = (Principal × Rate × Time) / 100
